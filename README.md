## 📱 Install as App

### On Mobile:
1. Open website in Chrome/Safari
2. Click menu → "Install app"
3. Works offline!

### On Desktop:
1. Click menu → "Install tenthie.s"
2. Works in standalone window

## 📊 Technologies Used

- HTML5 + CSS3 + Vanilla JavaScript
- LocalStorage for data persistence
- Service Worker for offline support
- PWA (Progressive Web App)
- GitHub Pages hosting

## 📝 How to Use

1. **Home:** View stats and announcements
